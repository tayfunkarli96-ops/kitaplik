### 🌐 Canlı API Bilgileri
- **Base Domain:** https://kitaplik-backend.vercel.app
- **Durum:** 200 OK (Aktif)

### ✅ Test Edilen Metotlar
1. **Profil Düzenleme:** PUT `/api/profile/1`
2. **Film Güncelleme:** PUT `/api/movies/1`
3. **Kadro Bilgileri:** GET `/api/movies/1/cast`
4. **Film Listeleme:** GET `/api/movies`
5. **Yorum Düzenleme:** PUT `/api/comments/1`
6. **İzlenecekler Listesi:** POST `/api/watchlist`
7. **Yorum Onaylama:** PUT `/api/comments/1/approve`
8. **Film Önerme (Quiz):** POST `/api/quiz/recommend`
9. **Haber Düzenleme:** PUT `/api/news/1`
10. **Dil Seçenekleri:** GET `/api/language/TR`
